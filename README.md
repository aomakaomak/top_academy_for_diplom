# top_academy_for_diplom

Учебный проект по математической статистике, теории вероятностей, визуализации данных и базовым элементам линейной алгебры. Репозиторий собран как практическая база для подготовки к дипломной работе: сначала темы разбираются в Jupyter-ноутбуках, затем ключевые идеи можно переносить в обычные Python-модули и покрывать тестами.

## О проекте

Проект сочетает два формата работы:

- **ноутбуки в папке `colab/`** — пошаговые практические занятия с пояснениями, кодом и `assert`-проверками;
- **обычный Python-код в `src/`** — место для вынесения переиспользуемой логики из ноутбуков в модули;
- **тесты в `tests/`** — проверка корректности функций;
- **документы в `docs/`** — сопроводительные материалы по проекту.

Судя по содержанию ноутбуков, проект покрывает базовый, но очень полезный для диплома набор тем: описательные статистики, выбросы, распределения, вероятности, формулу Байеса, доверительные интервалы, бутстрэп, а также вводные примеры по векторам и матрицам.

## Что изучается в ноутбуках

### `block04_lesson01_todo.ipynb`
Тема: **выборка, статистики и устойчивость к выбросам**.

Разбираются:
- размер выборки, минимум, максимум, сортировка;
- среднее и медиана;
- выборочная дисперсия и стандартное отклонение;
- влияние выбросов на среднее;
- усечённое среднее (`trimmed mean`);
- мини-описание выборки через функцию `describe(...)`.

### `block04_lesson02_todo.ipynb`
Тема: **распределения и гистограммы**.

Разбираются:
- генерация нормального распределения;
- расчёт среднего и стандартного отклонения через NumPy;
- построение гистограммы;
- влияние выброса на распределение;
- медиана;
- простая проверка правила 68%;
- сравнение нормального и равномерного распределений.

### `block04_lesson03_todo.ipynb`
Тема: **вероятность, условная вероятность, независимость и симуляции**.

Разбираются:
- представление событий в виде записей со значениями признаков;
- подсчёт частот;
- вероятность события `P(A)`;
- условная вероятность `P(A|B)`;
- приближённая проверка независимости;
- таблица сопряжённости `2x2`;
- визуализация условных вероятностей;
- симуляция кликов и покупок на случайных данных.

### `block04_lesson04_todo.ipynb`
Тема: **формула Байеса и наивный скоринг**.

Разбираются:
- подготовка бинарных счётчиков;
- вычисление вероятностей по частотам;
- likelihood, prior и evidence;
- формула Байеса;
- сравнение прямого подсчёта с байесовским обновлением;
- наивная оценка вероятности покупки по признаку;
- сглаживание Лапласа;
- простая визуализация изменения вероятности после обновления.

### `block04_lesson05_todo.ipynb`
Тема: **доверительные интервалы и бутстрэп**.

Разбираются:
- среднее и выборочное стандартное отклонение;
- стандартная ошибка среднего (`SEM`);
- приближённый 95% доверительный интервал;
- бутстрэп-оценивание среднего;
- bootstrap confidence interval через квантили;
- визуализация распределения бутстрэп-средних;
- сравнение ширины интервала на маленькой и большой выборке.

### `block04_lesson11_simple_todo.ipynb`
Тема: **векторы**.

Разбираются:
- создание векторов через `numpy.array`;
- длина вектора (`np.linalg.norm`);
- сложение векторов;
- скалярное произведение;
- простая визуализация векторов на плоскости.

### `block04_lesson12_simple_todo.ipynb`
Тема: **матрицы**.

Разбираются:
- создание матрицы;
- размерность через `.shape`;
- создание вектора весов;
- умножение матрицы на вектор;
- интерпретация результата в духе задач машинного обучения.

## Роли основных папок и файлов

### `colab/`
Основная учебная часть проекта. Здесь находятся ноутбуки с теорией, заданиями, демонстрациями и встроенными проверками.

### `src/`
Папка для Python-модулей. По названиям файлов и темам ноутбуков логика, вероятнее всего, делится так:

- `math_stats.py` — функции для описательной статистики, дисперсии, стандартного отклонения, доверительных интервалов и бутстрэпа;
- `gistogrammy.py` — работа с распределениями и построение гистограмм;
- `verojatnost.py` — вероятность, условная вероятность, таблицы сопряжённости и формула Байеса.

### `tests/`
Проверки для вынесенных в модули функций. Это важно, если код из ноутбуков постепенно переносится в производимую структуру проекта.

### `docs/plan.docx`
Документ с планом проекта, этапами работы или черновыми материалами по дипломной части.

### `data/.gitkeep`
Технический файл, который позволяет хранить пустую директорию `data` в Git.

## Как запустить проект

### 1. Создать и активировать виртуальное окружение

**Windows (PowerShell):**

```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

**Linux / macOS:**

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

По структуре проекта видно, что используются как минимум:
- `numpy`
- `matplotlib`
- `pytest`

### 3. Запустить тесты

```bash
pytest -q
```

### 4. Открыть ноутбуки

```bash
jupyter notebook
```

или

```bash
jupyter lab
```

После этого можно последовательно запускать ноутбуки из папки `colab/` и проверять результат через `Run All`.

## Практическая ценность проекта

Этот репозиторий хорошо подходит для следующих задач:

- повторить математическую статистику перед дипломом;
- собрать понятные примеры для пояснительной записки;
- перенести учебный код из ноутбуков в модули `src/`;
- научиться сопровождать расчёты тестами;
- подготовить наглядные графики и простые демонстрации для защиты.

## Что можно улучшить дальше

1. Перенести все ключевые функции из ноутбуков в `src/`.
2. Расширить тесты не только для статистики, но и для вероятностных модулей.
3. Добавить единый файл с примерами запуска модулей.
4. Переименовать файлы в едином стиле, если проект будет использоваться как портфолио.
5. Добавить примеры применения на реальных данных для дипломной темы.

## Итог

Проект уже выглядит как удобная учебная база: теория, практика, визуализация, модульная структура и тесты собраны в одном месте. Для дипломной подготовки это хороший формат, потому что позволяет не просто читать формулы, а сразу применять их в коде и проверять результат.
